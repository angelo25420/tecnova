Información para Usuarios

    Tecnova es un sistema web completo para gestionar ventas, productos, inventario y movimientos dentro de una tienda. Está diseñado para negocios que venden artículos de tecnología, limpieza, alimentos, papelería o similares.

    👤 Tipos de Usuario

    • Administrador: Control total del sistema (productos, entradas/salidas, historial, usuarios).
    • Usuario: Puede consultar el catálogo, ver productos disponibles y realizar ventas básicas.

    🛠️ Funcionalidades Clave

    • 📦 Gestión de Productos: Agregar, editar, eliminar productos con imagen, precio y cantidad.
    • 🛒 Carrito de Compras: Agrega productos al carrito y registra automáticamente en MySQL.
    • 📈 Historial de Movimientos: Registro automático de entradas, salidas y eliminaciones.
    • 🔍 Catálogo Visual: Navegación amigable con fotos y detalles de cada producto.
    • 📊 Reportes: Total de ventas, ingresos, productos más vendidos (opcional).

    🌐 Acceso

    El sistema se ejecuta localmente o en servidor web con conexión a MySQL y Firebase para mostrar imágenes en tiempo real.

    📌 Recomendaciones

    • ✅ Utiliza navegadores actualizados (Chrome, Edge, Firefox).
    • ⚠️ No cierres sesión sin guardar tus movimientos.
    • 💾 Realiza copias de seguridad de la base de datos cada semana.
